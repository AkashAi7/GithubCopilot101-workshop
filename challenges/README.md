## Challenges

These challenges are designed to make participants apply the intermediate GHCP features introduced in the labs rather than falling back to Ask mode alone.

## Challenge sequence

### Challenge 1 - Create a Custom MCP Agent

- Focus: combine BRD context, Figma MCP design context, and a 4-agent kit.
- Deliverable: an agent ecosystem that can answer citizen intake questions using tool-grounded context and explicit handoffs.
- File: [01-challenge-1-create-a-custom-mcp-agent.md](./01-challenge-1-create-a-custom-mcp-agent.md)

### Challenge 2 - Build a Public-Sector Copilot Playbook

- Focus: create reusable instructions, prompts, and a skill for consistent delivery, starting from Awesome Copilot resources.
- Deliverable: a portable playbook the team can reuse across future government application work.
- File: [02-challenge-2-build-a-public-sector-copilot-playbook.md](./02-challenge-2-build-a-public-sector-copilot-playbook.md)

### Challenge 3 - Orchestrate Delivery with Copilot and CLI

- Focus: decompose work, parallelize tasks, and prepare a fleet-style CLI execution plan using Spec Kit and the 4-agent ecosystem.
- Deliverable: a task board, agent ecosystem contract, execution prompts, and a thin runnable app slice created from the BRD.
- File: [03-challenge-3-orchestrate-delivery-with-copilot-and-cli.md](./03-challenge-3-orchestrate-delivery-with-copilot-and-cli.md)
