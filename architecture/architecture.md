## Architecture and Technology Stack

### Chosen Technologies
- **GitHub Copilot**: To assist developers in generating code snippets and improving productivity.
- **GitHub Copilot for MCP**: To create custom agents and workflows tailored to specific business needs.
- **skill.md**: To define and manage skills for custom agents.

### Rationale
- **GitHub Copilot** reduces development time by providing AI-driven code suggestions.
- **MCP** enables developers to create intelligent agents that can automate workflows and respond to specific scenarios.
- **skill.md** ensures modular and reusable skill definitions for agents.
